Unos ejemplos de tablas de evaluación. Las tablas incitan al alumnos a completar todas las actividades y, al proporcionar una nota, a hacerlo correctamente. Estas tablas leen los datos de las variables JavaScript de la página. Como el tiempo de vida de las variables JavaScript está lmitado al tiempo de vida de la sesión de la página html, cuando esta última se recarga los datos se pierden. Sin embargo, el usuario todavía puede saber que ya ha realizado una actividad con las tablas de resultados previos ya que sus datos se guardan en el web storage, una forma persistente de almacenar datos que el navegador vincula al sitio web de la página.
